## 二次型是什么

二次型通常写成：

```text
x^T A x
```

其中 `x` 是向量，`A` 是矩阵。

它会把一个向量映射成一个标量。很多机器学习里的目标函数和距离度量都可以写成二次型。

比如最简单的 L2 范数平方：

```text
||x||_2^2 = x^T x
```

这就是 `A = I` 时的二次型。

## 正定矩阵

如果对任意非零向量 `x`，都有：

```text
x^T A x > 0
```

那么矩阵 `A` 是正定矩阵。

如果允许等于 0：

```text
x^T A x >= 0
```

则是半正定矩阵。

正定性可以理解为：这个矩阵定义的二次曲面总是向上，不会出现负方向。

## 为什么优化关心正定

在优化里，Hessian 矩阵描述函数的局部曲率。

如果 Hessian 是正定的，函数在这个点附近像一个向上的碗，局部最小值更容易判断。

如果 Hessian 有负特征值，说明某些方向向下弯，可能是鞍点或局部最大附近。

深度学习的损失曲面很复杂，但“曲率”和“正定性”的语言仍然很有用。二阶优化、牛顿法、拟牛顿法都会用到这些概念。

## 协方差矩阵

协方差矩阵描述多维随机变量各维度之间如何共同变化。

对随机向量 `x`，协方差矩阵是：

```text
Sigma = E[(x - mu)(x - mu)^T]
```

它的对角线是每个维度的方差，非对角线是维度之间的协方差。

协方差矩阵一定是半正定的。因为对任意向量 `v`：

```text
v^T Sigma v
```

表示数据投影到方向 `v` 后的方差，而方差不可能是负数。

## 多元高斯与椭球

多元高斯分布常写成：

```text
x ~ Normal(mu, Sigma)
```

这里 `Sigma` 决定分布的形状。

如果 `Sigma` 是单位矩阵，分布在各个方向上扩散程度一样，像一个圆球。

如果不同方向方差不同，分布会拉成椭球。协方差矩阵的特征向量给出椭球主轴方向，特征值给出沿这些方向的扩散强度。

这就是线性代数和概率论在多元高斯里汇合的地方。

## 马氏距离

普通欧氏距离默认每个方向同等重要：

```text
||x - mu||_2
```

马氏距离会考虑协方差结构：

```text
sqrt((x - mu)^T Sigma^(-1) (x - mu))
```

如果某个方向本来方差很大，那么沿这个方向偏离一点不算太异常；如果某个方向方差很小，偏离一点就更显眼。

异常检测、聚类和高斯模型中经常会遇到这个思想。

## 正则化中的二次型

L2 正则可以写成：

```text
w^T w
```

更一般的正则可以写成：

```text
w^T A w
```

不同的 `A` 表示对参数不同方向施加不同惩罚。

这说明正则化不只是“让参数小”，也可以是“让参数在某些方向上更受限制”。

## 小结

二次型是连接几何、优化和概率的重要表达。正定矩阵保证二次型不为负，协方差矩阵天然半正定，多元高斯用协方差矩阵描述不确定性的形状。

理解这组概念，再看 PCA、牛顿法、高斯模型、异常检测和正则化，会更容易看到同一套数学结构。
