## 训练模型就是最小化损失

神经网络训练的目标可以简单写成：

```text
minimize loss(theta)
```

这里的 `theta` 表示模型参数，`loss` 表示模型在数据上的错误程度。

训练不是让模型“理解”某个抽象规则，而是不断调整参数，让损失函数变小。微积分和优化方法就是描述“怎么调整”的工具。

## 导数告诉我们变化率

一维函数里，导数表示函数在某一点的变化率。

如果：

```text
f'(x) > 0
```

说明 x 增大时，函数值在上升。

如果：

```text
f'(x) < 0
```

说明 x 增大时，函数值在下降。

训练模型时，我们关心的是参数变化会怎样影响 loss。导数给出了这个方向信息。

## 梯度是多维导数

模型参数不是一个数字，而是成千上万甚至上千亿个数字。

多维情况下，导数扩展成梯度：

```text
grad loss(theta)
```

梯度指向函数上升最快的方向。既然我们想让 loss 变小，就应该沿着梯度的反方向走：

```text
theta = theta - learning_rate * grad
```

这就是梯度下降的核心公式。

## 学习率决定步子大小

学习率控制每次更新走多远。

学习率太大，可能越过最低点，训练震荡甚至发散。学习率太小，训练会很慢，也可能卡在不理想的位置。

所以很多训练技巧都围绕学习率展开：

- warmup
- cosine decay
- step decay
- adaptive optimizer

它们的目的都是让模型在不同训练阶段用更合适的步子移动。

## 链式法则与反向传播

神经网络由很多层函数复合而成：

```text
y = f3(f2(f1(x)))
```

链式法则告诉我们，复合函数的导数可以一层层相乘。

反向传播就是链式法则在神经网络里的工程实现。它从 loss 开始，沿着计算图反向传递梯度，计算每个参数应该如何调整。

这也是为什么深度学习框架需要记录计算图。PyTorch 的 `loss.backward()` 本质上就是自动完成这件事。

## 局部最优不是唯一问题

很多人刚学优化时会担心局部最优。但在深度学习里，优化难点不只是局部最优，还包括：

- 梯度消失
- 梯度爆炸
- 鞍点
- 噪声很大的 mini-batch 梯度
- 参数规模巨大导致的训练不稳定

这也是为什么现代训练会结合归一化、残差连接、梯度裁剪、AdamW、学习率调度等技巧。

## 小结

导数描述变化率，梯度描述多维参数空间里的最快上升方向，梯度下降沿着反方向降低损失，反向传播用链式法则高效计算所有参数的梯度。

理解这条链路，训练过程就不再像黑盒：模型每一步都在回答同一个问题，参数应该往哪个方向改，loss 才会更小。
