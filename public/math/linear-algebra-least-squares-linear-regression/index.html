线性代数：最小二乘与线性回归

2026-07-10 数学 线性代数 线性代数 最小二乘 线性回归 投影 机器学习

为什么从线性回归学线代

线性回归是最适合把线性代数和机器学习连起来的模型。

它足够简单，但包含了很多关键思想：矩阵表示、参数估计、损失函数、投影、正则化和泛化。

如果理解了线性回归里的线代结构，再看更复杂的神经网络线性层、注意力投影和 embedding 变换，会轻松很多。

把数据写成矩阵

假设有 n 个样本，每个样本有 d 个特征。

可以把所有样本堆成一个矩阵：

X: [n, d]

标签写成向量：

y: [n]

线性模型写成：

y_hat = Xw

这里 w 是要学习的参数。

最小二乘目标

线性回归希望预测值接近真实值。最常见的目标是平方误差：

min_w ||Xw - y||_2^2

这个式子叫最小二乘。

从机器学习角度看，它是经验风险最小化；从概率角度看，如果假设观测噪声是高斯分布，它也对应最大似然估计。

几何直觉：投影

Xw 能表示什么？

它表示 X 的列向量的线性组合。也就是说，所有可能的预测 Xw 都落在 X 的列空间里。

如果真实标签 y 不在这个列空间里，就无法完全拟合。最小二乘会找一个列空间里的点，让它离 y 最近。

这个最近点就是 y 在列空间上的投影。

所以最小二乘不只是优化问题，也是几何投影问题。

正规方程

对目标函数求导并令梯度为 0，可以得到正规方程：

X^T X w = X^T y

如果 X^T X 可逆，解是：

w = (X^T X)^(-1) X^T y

实际工程里通常不会显式求逆，因为数值稳定性和计算效率都不理想。更常用的是梯度下降、QR 分解、SVD 或成熟线性代数库。

但正规方程仍然值得理解，因为它展示了最小二乘的结构。

欠定、超定与秩

如果样本数多于特征数，方程通常是超定的，不能完全满足所有样本，只能找最小误差解。

如果特征数多于样本数，方程可能是欠定的，会有很多解。

矩阵的秩决定了特征中有多少独立信息。特征高度相关时，X^T X 可能病态，参数会变得不稳定。

这就是为什么线性回归里会讨论多重共线性。

岭回归与稳定性

岭回归在目标里加入 L2 正则：

min_w ||Xw - y||_2^2 + lambda ||w||_2^2

对应的方程变成：

(X^T X + lambda I)w = X^T y

lambda I 会让矩阵更稳定，也会限制参数大小。它牺牲一点训练误差，换来更好的泛化和数值稳定性。

小结

最小二乘把线性回归写成矩阵优化问题。它的核心直觉是：在特征矩阵的列空间里，找离真实标签最近的投影。

线性回归看似简单，但它把表示、优化、概率假设和正则化都串起来了。
