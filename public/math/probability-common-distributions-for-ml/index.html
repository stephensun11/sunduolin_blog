概率论：机器学习里常见的概率分布

2026-07-10 数学 概率论 概率论 分布 高斯分布 伯努利分布 机器学习

为什么要认识分布

概率分布不是为了背公式。对机器学习来说，分布回答的是：我们对数据、标签、噪声、参数或动作的不确定性，打算用什么形状来描述。

选错分布，模型假设就会别扭。比如二分类标签天然适合伯努利分布；连续误差常用高斯假设；多类别标签常用 Categorical 分布。

这篇只整理最常见、最容易在机器学习里遇到的分布。

伯努利分布：一次二选一

伯努利分布描述一次 0/1 事件：

X ~ Bernoulli(p)

X = 1 的概率是 p，X = 0 的概率是 1 - p。

机器学习里的典型场景：

  • 二分类标签。
  • 一个 token 是否命中某个条件。
  • 一个样本是否被模型判对。
  • dropout 中某个神经元是否保留。

二分类交叉熵可以理解为伯努利分布下的负对数似然。

Categorical 分布：一次多选一

Categorical 分布描述多类别中的一次选择。

比如三分类模型输出：

类别 概率
猫 0.70
狗 0.20
鸟 0.10

这就是一个 Categorical 分布。

语言模型对词表中下一个 token 的预测，也是 Categorical 分布。softmax 的作用，就是把一组 logits 转成一个合法的概率分布。

p_i = exp(z_i) / sum_j exp(z_j)

二项分布：多次伯努利

二项分布描述 n 次独立 0/1 实验里，成功次数是多少：

X ~ Binomial(n, p)

它适合理解“重复试验的命中次数”。例如模型在一批样本上的正确个数，可以粗略看成二项分布。当然真实训练中样本不完全独立，难度也不完全相同，但这个视角有助于理解准确率的波动。

如果 batch 很小，正确率上下跳动会很明显；batch 变大，统计量会更稳定。

泊松分布：单位区间里的计数

泊松分布常用于描述一段时间或空间内发生的次数：

X ~ Poisson(lambda)

它适合稀疏计数场景，比如某类事件的出现次数、点击流中的稀有行为、日志中的异常告警数量。

在推荐、广告和风控里，经常会遇到计数型数据。理解泊松分布，有助于判断什么时候不应该直接用高斯误差去套所有问题。

高斯分布：连续变量与噪声

高斯分布也叫正态分布：

X ~ Normal(mu, sigma^2)

它在机器学习里出现得非常频繁：

  • 线性回归常假设误差服从高斯分布。
  • 初始化权重时常从高斯或近似高斯分布采样。
  • VAE 中会让潜变量接近标准高斯。
  • 扩散模型会不断添加和去除高斯噪声。

高斯分布的两个参数很直观：mu 控制中心位置，sigma^2 控制分散程度。

多元高斯：协方差开始重要

当随机变量变成向量时，就有了多元高斯：

x ~ Normal(mu, Sigma)

这里 mu 是均值向量，Sigma 是协方差矩阵。

协方差矩阵描述不同维度如何一起变化。PCA、马氏距离、高斯混合模型都离不开这个对象。

如果只记一句话：一维高斯看均值和方差，多维高斯看均值向量和协方差矩阵。

Dirichlet 分布：分布上的分布

Dirichlet 分布可以理解为“概率向量的分布”。它生成的结果本身是一组和为 1 的概率。

这在主题模型里很常见。LDA 会把一篇文档看成多个主题的混合，而主题比例就是一个概率向量。

Dirichlet 不需要一开始就掌握所有推导，但要知道它解决的是这类问题：我们不仅关心某个类别的概率，还关心“类别概率这组参数本身的不确定性”。

小结

分布是模型假设的一部分。伯努利用于二选一，Categorical 用于多选一，二项用于重复命中次数，泊松用于计数，高斯用于连续噪声，多元高斯引入协方差，Dirichlet 描述概率向量。

学这些分布，不是为了背密度函数，而是为了看到模型在用什么方式描述不确定性。
