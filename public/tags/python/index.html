Python 实用入门：把重复工作写成脚本

2026-07-06 CSTech Python Python脚本自动化开发基础

Python 最适合作为第一门实用脚本语言来学。 它的语法不重，标准库够丰富，写几行代码就能处理文件、整理数据、调用命令、生成报告。对个人博客、AI 实验和日常开发来说，Python 很像一把随手能拿起来的小工具。 这篇文章不追求讲完 Python，而是帮你建立一个能立刻动手的最小知识框架。 安装与版本检查 安装完成后，先确认 Python 是否可用。 python --version 有些系统里命令叫 python3。 python3 --version 进入交互环境： python 执行脚本： python hello.py 一个最小脚本如下。 print("Hello, Python") 虚拟环境 写 Python 项目时，建议每个项目使用独立虚拟环境。这样不同项目的依赖不会互相污染。 创建虚拟环境： python -m venv .venv 激活虚拟环境。 Windows PowerShell： .\.venv\Scripts\Activate.ps1 Linux / macOS： source .venv/bin/activate 安装依赖： pip install requests 导出依赖： pip freeze > requirements.txt 下次恢复环境： pip install -r requirements.txt 变量和基本类型 Python 里常见的数据类型很直接。 name = "Alice" age = 18 score = 95.5 is_active = True tags = ["AI", "Python", "Blog"] profile = {"name": name, "age": age} 列表适合保存一组有顺序的数据。
